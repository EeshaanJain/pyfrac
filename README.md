# pyfrac
A python library to work with Fractions

```bash
pip install pyfrac
```

# Examples
```python
>>> from pyfrac import Fraction as F                                                                                   
>>> f1 = F(1, 2)                                                                                                        
>>> f2 = F(2, 3)                                                                                                        
>>> f3 = f1 + f2                                                                                                        
>>> f3                                                                                                                  
Fraction(numerator: 7, denominator: 6)                                                                                  
>>> print(f3)                                                                                                           
7 / 6             
```
